Pranay Theja Bandarkanti — Portfolio
A personal portfolio website built with pure HTML, CSS, and WebGL. Features a animated nebula background, bento-grid layout, and sections for projects, skills, education, and contact.
🌐 Live Preview
> Deploy on GitHub Pages, Vercel, or Netlify — just upload `index.html`.
✨ Features
Animated WebGL background — real-time GLSL shader rendered on a canvas
Sticky glassmorphism navbar with smooth scroll navigation
Bento grid layout for projects, skills, and about sections
Fully responsive — mobile-friendly with a stacked layout on small screens
Zero dependencies — single HTML file, no build tools or npm required
Only external resource: Tabler Icons via CDN
📁 Project Structure
```
portfolio/
└── index.html   # Entire site — HTML + CSS + JS in one file
```
🛠️ Tech Stack
Layer	Technology
Markup	HTML5
Styling	CSS3 (custom, no framework)
Background	WebGL 2.0 · GLSL shaders
Icons	Tabler Icons (CDN)
Fonts	System UI (no external font loading)
🚀 Getting Started
Run locally
No setup needed. Just open the file in a browser:
```bash
# Clone the repo
git clone https://github.com/pranay070605/<repo-name>.git

# Open in browser
open index.html
```
Deploy to GitHub Pages
Push `index.html` (rename it to `index.html` if needed) to your repository
Go to Settings → Pages
Set source to `main` branch, `/ (root)`
Your site will be live at `https://pranay070605.github.io/<repo-name>/`
